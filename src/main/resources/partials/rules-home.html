Match Predictions and points
  • User can predict the score for each match just before the match starts. Default prediction set for each match is 0-0 when registered.
  • Scoring is as follows:
    • If exact match = 30 points
    • If goal difference same (and winner correct) = 20 points
    • If winner correct = 10 points
    • For Round of 16, the above points will be multiplied by 2 (2x) For League tournament, not applicable
    • For Quarter Finals, the above points will be multiplied by 3 (3x) For League tournament, not applicable
    • For Semi Finals, Play-off for Third Place and Finals, the above points will be multiplied by 4 (4x)
  • In case of knockout games only the final score (without the penalty shootout) will be considered.
  • Addition: Any new user created after tournament started will have the previous Match predictions as 0-0 and will get point based on 0-0 only.
Global Predictions
  • All the users will be allowed to predict the winner, runner-up and total goals in the tournament.
  • This can be updated just before the elimination round starts. After that no changes will be supported.
  • The points for team predictions will be as follows:
    • Exact Winner and Runner up = 150 points
    • Teams of final correct = 100 points
    • Only Winner correct = 75 points
    • Only Runner up correct = 60 points
    • One team of final is correct = 50 points
  • The total goals count will not include the penalty shoot out score in knockout stage.
  • The goal difference for each user will be calculated and the ones with the closest will be awarded highest points.
  • Closest ones will get = 100 points
  • Second closest will get = 60 points
Prize distribution
  • All transactions in cash only. :)
  • The total money will be divided into the top 3 scores in ratio 5:3:2
  • This means there can be multiple users at the same score and they all will receive the same money but in the ratio defined as above.
